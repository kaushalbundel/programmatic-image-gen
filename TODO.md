## Additional thoughts about the project
### Front end
      - Vue gives two options: Option API and composition API. The general recommendation with these api's is:
         - If vue is to be used without a build step meaning wherever vue is strictly needed option api should be used
         - If vue is to be used with a build step then composition api is preferred.

      - I am going to use vue for creating the frontend. The backend would be handled separately by python. So, I should use the build step and should use composition API.
